Post-Installation

After running the installation script:

1. Secure Boot Users

If Secure Boot is enabled, you'll need to enroll the MOK (Machine Owner Key). The script will guide you through this process.

2. Reboot Required

Reboot your system to load the new driver. Your WiFi adapter should be recognized automatically.

3. Verify Installation

Check that the driver is loaded with: lsmod | grep rtw
